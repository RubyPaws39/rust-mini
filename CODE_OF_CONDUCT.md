# Code Of Conduct

Rust Mini should be a friendly learning project.

## Expected Behavior

- be kind
- be patient with beginners
- explain technical feedback clearly
- focus criticism on code and ideas
- welcome questions

## Unacceptable Behavior

- harassment
- insults
- personal attacks
- gatekeeping
- spam
- hostile review comments

## Reporting

Open a private message or issue with the maintainer if something needs attention.

Maintainers may remove comments, close issues, or block users to keep the project safe and useful.
